## Recommendations

1. **Fix keyword search (DN-BUG-001):** Ensure `ma_doanh_nghiep` is included in the search index. Users will search by DN code frequently.
2. **Clarify edit page UX (DN-BUG-002):** Either make the edit URL lead to editable fields, or change the URL pattern to `/chi-tiet` for view and `/chinh-sua` for edit mode.
3. **Implement Import Excel (DN-BUG-003):** SRS marks this as Essential. Prioritize for next sprint.
4. **Manual re-test needed:** DN-004 (MST unique), DN-007 (delete guard with active VV).
5. **Cross-module test:** Create a VV linked to a DN, then test DN-007 guard rule.
