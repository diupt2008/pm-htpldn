### TVV-007: Xem chi tiết TVV (P0)

| Field | Value |
|-------|-------|
| **Status** | **FAIL** |
| **Severity** | Medium (automation issue, not product bug) |

**Problem:** Clicking the first table row timed out after 30s. The `<tr>` element resolved to `<tr aria-hidden="true">` which is a virtual scroll placeholder, not a real data row. The table uses virtualized rendering.

**Root cause:** The `.ant-table` in this module uses row virtualization (aria-hidden rows). Standard `tbody tr:first-child` selects the placeholder instead of the actual visible row.

**Bug TVV-BUG-001:** Table action column click pattern needs investigation. Either:
- Action icons are inside a dropdown menu (need hover first)
- Row click navigation requires clicking the TVV name/code link specifically
- Virtual scroll rows need different selectors

**Impact:** All detail page tests (TVV-007 through TVV-022 workflow) are blocked.
