## Ghi chú đặc thù module
- Dashboard hiển thị 2 stat card `Đào tạo đang diễn ra` / `Đào tạo hoàn thành` — dùng để quick check data readiness.
- **Known bug BUG-DT-01:** navigate từ `/tao-moi` → `/danh-sach` có thể hiển thị toast `Validation failed (uuid is expected)` + content `Không tìm thấy khóa học`. Nếu gặp ở Bước 3 → note vào report, không block verdict (đã reported).
- **Known state 2026-04-18:** DB trống — 0 khóa học, nghi 0 CTDT. Bước 2 sẽ hiển thị `Không có khóa học nào phù hợp.` — coi là PASS nếu tabs + nút `Thêm mới` hiện đủ.
- Submenu `Quản lý đào tạo, tập huấn` có 4 item con: `Chương trình đào tạo` / `Khóa học` / `Ngân hàng câu hỏi` / `Giảng viên`.

  | Response | Action |

## Bước 2: VÀO MODULE

### 2a. Verify menu + submenu sidebar

1. `$B snapshot -i`
2. Grep `Quản lý đào tạo, tập huấn` (có ▶) — phải xuất hiện
3. Click menu → đợi expand (submenu 4 item)
4. Snapshot lại → grep `Khóa học` (phải thấy trong submenu)

| Kết quả | Action |
|---------|--------|
| ✅ Menu + submenu `Khóa học` visible | → 2b |
| ❌ Menu không thấy | Verify role account. Thiếu → **FAIL**, báo FE |
| ❌ Submenu không expand | Retry click 1 lần. Vẫn không → **FAIL**, báo FE |

### 2b. Navigate vào Khóa học

1. Click submenu `Khóa học` (hoặc `$B goto http://103.172.236.130:3000/dao-tao/khoa-hoc/danh-sach` trực tiếp — stable hơn)
2. Đợi page render (< 5s)
3. Verify URL = `/dao-tao/khoa-hoc/danh-sach` + 10 tabs + nút `Thêm mới`
4. Chụp `daotao-page.png`

| Kết quả | Action |
|---------|--------|
| ✅ URL + elements | → Bước 3 |
| ❌ Redirect `/login` | Session lỗi, báo dev |
| ❌ Page trắng >10s | **FAIL**, báo BE |
| ❌ Thiếu tabs/Thêm mới | **FAIL**, note |

## Bước 3: KIỂM TRA LỖI NGẦM

1. `$B console --errors`
2. `$B network` → grep 4xx/5xx
3. `$B snapshot` → grep `"Lỗi"`, `"Validation failed"`, `"Không thành công"`, `"undefined"`

**Lưu ý BUG-DT-01:** Nếu thấy toast `Validation failed (uuid is expected)` + `Không tìm thấy khóa học` ở /danh-sach → đây là known bug route state, note vào report, **không block PASS**.

| Kết quả | Verdict |
|---------|---------|
| Sạch (hoặc chỉ BUG-DT-01) | ✅ PASS |
| Warning deprecation / 401 single | ⚠️ WARN |
| TypeError / 5xx / toast đỏ lạ (không phải BUG-DT-01) | ❌ FAIL |
