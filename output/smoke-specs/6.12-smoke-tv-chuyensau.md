## Ghi chú đặc thù module
- **Permission:** CB NV TW **có** Create. CB PD **KHÔNG** tạo được TVCS. CG/TVV chỉ xác nhận/trả lời khi được phân công.
- **Precondition data:**
  - ≥1 Doanh nghiệp
  - ≥1 TVV/CG state `DANG_HOAT_DONG` (phụ thuộc module 7.4 — smoke 6.4)
  - ≥1 lĩnh vực PL
  - Nếu smoke 6.4 (TVV) đã PASS + đã tạo data → precondition có sẵn.
- **Detail view** có stepper 6 bước — ngoài scope smoke Bước 2-3.
- **Rich Text Editor** nội dung tư vấn — nếu test nâng cao, dùng `$B type` trên `[contenteditable="true"]` hoặc `.ql-editor`.

## Bước 2: VÀO MODULE

### 2a. Verify menu + submenu sidebar

1. `$B snapshot -i`
2. Grep `Quản lý tư vấn` (có ▶) — phải xuất hiện
3. Click menu → đợi submenu expand
4. Snapshot lại → grep `Tư vấn chuyên sâu`

| Kết quả | Action |
|---------|--------|
| ✅ Menu + submenu `Tư vấn chuyên sâu` visible | → 2b |
| ❌ Menu không thấy | Verify role. Thiếu → **FAIL**, báo FE |
| ❌ Submenu không expand / không có `Tư vấn chuyên sâu` | Retry 1 lần. Vẫn không → **FAIL**, báo FE |

### 2b. Navigate

1. Click submenu `Tư vấn chuyên sâu`
2. Đợi page render (< 5s)
3. Verify URL + 3 tab nhóm + nút `+ Thêm yêu cầu TV`
4. Chụp `tvcs-page.png`

| Kết quả | Action |
|---------|--------|
| ✅ URL + elements | → Bước 3 |
| ❌ Redirect `/login` | Session lỗi, báo dev |
| ❌ Page trắng | **FAIL**, báo BE |
| ❌ Thiếu tabs/buttons | **FAIL**, note |

## Bước 3: KIỂM TRA LỖI NGẦM

1. `$B console --errors`
2. `$B network` → grep 4xx/5xx
3. `$B snapshot` → grep `"Lỗi"`, `"Validation failed"`, `"Không thành công"`, `"undefined"`

| Kết quả | Verdict |
|---------|---------|
| Sạch | ✅ PASS |
| Warning / 401 single | ⚠️ WARN |
| TypeError / 5xx / toast đỏ | ❌ FAIL |
