## Bước 2: VÀO MODULE

### 2a. Verify menu + submenu sidebar

1. `$B snapshot -i` (hoặc MCP `take_snapshot` — xem CLAUDE.md tool routing)
2. Grep `Quản lý tư vấn` (có ▶) — phải xuất hiện
3. Click menu → đợi submenu expand
4. Snapshot lại → grep `Tư vấn pháp luật chuyên sâu` `[RENAMED v3.5 Thay đổi 1]`

| Kết quả | Action |
|---------|--------|
| ✅ Menu + submenu `Tư vấn pháp luật chuyên sâu` visible | → 2b |
| ⚠️ Submenu vẫn ghi `Tư vấn chuyên sâu` (chưa đổi tên) | **FAIL** v3.5 — log bug FE chưa apply Thay đổi 1 |
