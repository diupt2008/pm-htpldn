## Ghi chú đặc thù module
- **Precondition data:** HS chi trả BẮT BUỘC tạo từ DVC/LGSP (integration external) → state ban đầu `CHO_TIEP_NHAN`. CB NV KHÔNG có nút `+ Thêm mới` trên SCR-V.II-01 (negative test).
- **Module BLOCKED end-to-end** do thiếu integration LGSP/DVC. Smoke chỉ verify list render + tab + filter + permission.
- **Số tiền lớn:** UI format theo VND với dấu chấm hàng nghìn + hậu tố "đ" (SRS v3.5 SCR-V.II-01 #13). Verify không hiển thị `NaN` / `undefined`.
- **Thanh toán external:** state `DA_THANH_TOAN` — integration với hệ thống kho bạc / ngân hàng, ngoài scope smoke.
- **CB PD "Từ chối" = trả về thẩm định (KHÔNG phải Từ chối cuối)** — verify nút "Từ chối — trả về thẩm định" ở SCR-V.II-02 #28 nếu test deep.
