## Bối cảnh

Module Tư vấn Nhanh là hệ thống tra cứu Q&A pháp lý (keyword search, KHÔNG AI) phục vụ DN tự tra cứu trên Cổng PLQG + cho phép CB NV gợi ý trả lời từ kho dựng sẵn với quy trình:
**DN gửi câu hỏi → Keyword search kho → Gợi ý TOP 5 → CB NV chọn/chỉnh sửa → Gửi trả lời → DN đánh giá**

**Kho câu hỏi có 3 nguồn bổ sung (BR-FLOW-10):**
1. `TU_DONG` — HOI_DAP (Nhóm II) DA_DUYET → tự động vào kho, KHÔNG cần duyệt lại
2. `THU_CONG` — CB NV nhập → CHO_DUYET → CB PD duyệt
3. `IMPORT` — Upload Excel → CHO_DUYET → CB PD duyệt

**Công khai (FR-X.2-06 mới v3.5):** Sau khi Q&A đạt DA_DUYET, CB NV có thể bật Switch [Công khai] → BE gọi API ra Cổng PLQG (BR-FLOW-05) → khi API thành công, `trang_thai` chuyển CONG_KHAI + auto-fill `thoi_gian_dang_tai`. Hủy công khai → CONG_KHAI → DA_DUYET + clear timestamp.

## State Machine — SM-TVNHANH (phiên TV)

```
MOI → DANG_TIM_KIEM → DA_GOI_Y → CB_TRA_LOI → HOAN_THANH
                         ↓
                      HOAN_THANH (DN hài lòng, đánh giá trực tiếp)
 ↓
HET_HAN (auto 30 ngày không xử lý)
```

**Trạng thái phiên TV nhanh:**

| Trạng thái | Mô tả |
|-----------|-------|
| MOI | DN vừa gửi câu hỏi qua Cổng PLQG |
| DANG_TIM_KIEM | HT đang keyword search kho Q&A |
| DA_GOI_Y | HT đã gợi ý TOP 5 câu trả lời |
| CB_TRA_LOI | CB NV đang/đã soạn + gửi trả lời |
| HOAN_THANH | DN đã đánh giá, kết thúc phiên |
| HET_HAN | Quá 30 ngày không xử lý (batch job) |

## State Machine — KHO_CAU_HOI (kho Q&A)

```
NHAP ← (CB PD từ chối + lý do) ← CHO_DUYET → DA_DUYET → CONG_KHAI
                                              ↑           ↓
                                              ←──── (Hủy công khai) ←──
                                                     ↓
                                                HET_HIEU_LUC
```

**Trạng thái KHO_CAU_HOI** (SRS line 689 + 528 + 534):

| Trạng thái | Mô tả |
|-----------|-------|
| NHAP | Bị CB PD từ chối, CB NV chỉnh lại + gửi duyệt lần 2 |
| CHO_DUYET | Default khi CB NV tạo thủ công / Import Excel |
| DA_DUYET | CB PD duyệt OK, hiệu lực = true mặc định, sẵn sàng trả lời TVN |
| CONG_KHAI | Đã đẩy lên Cổng PLQG thành công (kết quả của FR-X.2-06) |
| HET_HIEU_LUC | CB NV toggle off hiệu lực hoặc bản ghi không còn dùng |
