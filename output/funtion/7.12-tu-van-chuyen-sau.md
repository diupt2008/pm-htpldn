## Bối cảnh

Module Tư vấn Chuyên sâu xử lý vòng đời yêu cầu tư vấn 1-1 giữa DN ↔ Chuyên gia/TVV với quy trình:
**Tiếp nhận → Phân công CG → CG xác nhận → Đang tư vấn → Hoàn thành → Phê duyệt → Đã duyệt (gửi DN)**

Module có 4 UC loại B (CMS browser) + 3 UC loại M (API inbound từ Cổng PLQG):
- **CMS:** Quản lý nội dung TVCS (UC147), Tìm kiếm (UC148), Quản lý hồ sơ pháp lý DN (UC150), Quản lý tư liệu pháp lý vụ việc (UC152)
- **API inbound:** Tiếp nhận TVCS (UC149), Tiếp nhận HSPL (UC151), Tiếp nhận đánh giá chất lượng (UC153)

## State Machine — SM-TVCS

```
TIEP_NHAN → PHAN_CONG → DANG_TU_VAN → HOAN_THANH → CHO_PHE_DUYET → DA_DUYET
   ↓           ↓ ↑           ↓                          ↓
  HUY        HUY (CG từ chối)  HUY (DN+CB PD duyệt)  DANG_TU_VAN (từ chối)
              ↑
        TIEP_NHAN (timeout 2 ngày LV → auto-reject)
```

**Trạng thái:**

| Trạng thái | Mô tả |
|-----------|-------|
| TIEP_NHAN | YC TV mới tạo / tiếp nhận từ Cổng |
| PHAN_CONG | Đã phân công CG/TVV, chờ xác nhận |
| DANG_TU_VAN | CG đang tư vấn (sau xác nhận) |
| HOAN_THANH | CG hoàn thành (kèm VB TVPL) |
| CHO_PHE_DUYET | Auto-transition, chờ CB PD duyệt |
| DA_DUYET | Đã duyệt → gửi KQ cho DN, mở đánh giá |
| HUY | Hủy yêu cầu |
