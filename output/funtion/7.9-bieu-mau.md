## Bối cảnh

Module quản lý kho biểu mẫu/hợp đồng mẫu (HĐ lao động, HĐ chuyển nhượng, HĐ DV Marketing...) cho DNNVV tham khảo + tải về qua Cổng PLQG. Cấu trúc 2 cấp: **Thư mục → Biểu mẫu**. Công khai trực tiếp lên Cổng PLQG **KHÔNG cần phê duyệt** (BR-FLOW-07).

**Quy trình:** Tạo Thư mục → Thêm Biểu mẫu (upload file) → Công khai lên Cổng → DN tải về qua API. Ngoài ra có CRUD HĐ Tư vấn (FR-VII-08) liên kết vụ việc.

## State Machine — SM-BIEUMAU

```
     ┌───────── CONG_KHAI ◄──────┐
     │              ▲            │
NHAP ┤              │            │
     │              ▼            │
     └───────────── AN ──────────┘
          │                 │
          ▼                 ▼
         XOA               XOA
```

- NHAP → CONG_KHAI: công khai (không cần duyệt — BR-FLOW-07)
- CONG_KHAI ↔ AN: ẩn / công khai lại
- NHAP / AN → XOA: soft-delete (chỉ khi thư mục rỗng)

## Ghi chú thực thi

- **Authorization (BM-032 → BM-036):** Smoke-level. Full coverage ở [permission-matrix.md](../permission-matrix.md).
- **API Cổng PLQG (BM-038):** Dùng Postman/Bruno gọi trực tiếp endpoint. Verify chỉ trả về record state CONG_KHAI, filter hoạt động.
- **File test (BM-004, BM-015, BM-016):** Chuẩn bị sẵn:
  - `sample-valid.docx` (≤20MB) — happy path
  - `sample-oversize.docx` (>20MB) — test limit
  - `sample-invalid.txt` / `sample.exe` — test format
  - `sample-macro.docm` — test virus scan (EC-02)
- **Preview (BM-007):** Verify cả doc/docx (convert → PDF) và xls/xlsx (bảng read-only). Nếu không hỗ trợ preview → fallback "Tải về".
- **Import mixed (BM-029):** Upload 5 file = 3 hợp lệ + 1 >20MB + 1 sai format → verify 3 thành công, 2 lỗi có chi tiết.
- **HĐ Tư vấn (BM-030):** Cần seed TVV state DANG_HOAT_DONG trước (phụ thuộc module 7.3).
- **Audit (BM-040):** Verify trên màn Nhật ký HT — action codes: `CREATE_BIEU_MAU`, `UPDATE_BIEU_MAU`, `DELETE_BIEU_MAU`, `PUBLISH`, `UNPUBLISH`, `TAI_BIEU_MAU`, `BULK_IMPORT`.
- **Công khai (BM-022):** Verify cả trạng thái DB (CONG_KHAI) và record thực sự appear trên Cổng PLQG (qua API call BM-038).
